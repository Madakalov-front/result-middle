Разобраны ключи(key) для компоннетов, которые дают реакту понять в каком порядке следуют элементы для управления их отображением, удалением, добавлением.

Разобрана важность иммутабльность значений для корректной работы React.

Изучены методы классового компонента по управлению его состоянием жизненого цикла.

Разобран условный рендер, а так же особенности использования оператора &&. Switch, if - esle, - ? + : -, &&. Для себя подчеркнул присвоение компонента в переменную, что делает структуру jsx более лаконичной, но возможно немного усложняет восприятие.
